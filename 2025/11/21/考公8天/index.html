一、知彼知己，百战不殆

申论我已经做了，接下来是行测

政治理论：阅兵武器一道题 最新会议一道题，可能求是一道题

各种效应了解一下

如果离考试只剩 5 天左右，此时不适合大规模攻克新知识，而是要 迅速提升分数的“最短路径”。下面给你一个 按天安排、可直接执行的冲刺计划，非常实用👇

✅ 总原则：三句话

  1. 行测“练+总结”为主，申论“看范文+熟套路”为主。
  2. 不再大量背知识点，不刷难得离谱的大题。
  3. 只做最可能提分、最能稳定发挥的内容。

🗓️ 5 天冲刺计划（可直接照做）

📅 第 1 天：全科摸底 + 查漏补缺

目标：找到最近 5 天该干的重点。

行测

  • 做一套 和你考试年份相同的近三年真题（最好整套计时）。

  • 记录错误点：

    • 数量关系错的类型
    • 逻辑判断的薄弱点
    • 资料分析计算速度
    • 言语的主旨题/语句填空
    • 常识判断的领域短板

申论

  • 看一篇最近年份的真题答案与解析。
  • 观察：给分点格式、作答模板、大作文结构。

👉 目的：明确这 5 天的核心提分点，而不是盲目复习。

📅 第 2 天：加强行测“最可提分部分”

按照你的薄弱项来：

1. 资料分析（最容易短期提分）

  • 刷 2～3 套资料分析专项
  • 学 3～5 个常用快速算法（如基期法、增长率、差值法等）
  • 目标：正确率提升 + 计算速度提升

2. 判断推理（图形、逻辑）

  • 刷近年真题里的图形推理
  • 错题类比推理、逻辑判断再做 20～30 道

申论（轻量）

  • 学 2 个实用大作文模板
  • 看 2 篇高分范文，模仿其开头、结尾、分论点写法

📅 第 3 天：稳定言语 + 数量关系

1. 言语理解（容易在短期提升稳定性）

  • 主旨题 30 题

  • 语句排序 15 题

  • 语句填空 20 题

  • 错题记录规律

    • 主旨靠首段总结法
    • 排序靠逻辑连接词
    • 填空靠常用高频词对比

2. 数量关系（能做就做，不能就选概率高的）

如果你基础弱：
→ 只学常考模板题，不碰复杂题
如果基础中等：
→ 刷近 3 年真题数量关系题
目标：多得 2～5 分就很关键。

申论（轻量）

  • 做一套小题（概括、分析题）
  • 练 1 次 800 字作文提纲（不必全文写）

📅 第 4 天：整套模拟（最核心的一天）

务必完整做一套真题 + 写 1 套申论。
完全还原考试节奏（行测 120 分钟，申论 150 分钟）。

做完后必须做：

  • 找耗时最长的模块 → 有针对性优化
  • 分析错误原因
  • 行测重新做错题
  • 申论根据解析自我打分

👉 这是短期提升最大的一天。

📅 第 5 天：查漏补缺 + 稳定心态

前一天模拟暴露的问题，这一天全部解决。

行测

  • 资料分析刷 10 题
  • 图形推理刷 10 题
  • 主旨题刷 20 题
  • 数量关系刷 5～10 个熟悉题型

申论

  • 记住：

    • 小题模板
    • 大作文框架
    • 常用论证句、过渡句

  • 用 30 分钟写一篇大作文提纲即可

晚上做：

  • 减轻心理负担
  • 不再硬背知识点
  • 保持手感即可

🎯 最后一天早上（考试当天）

  • 看模板（申论）
  • 看自己易错点（行测）
  • 不再做新题

言语理解

1，阿里木江（理论基础课）

当然可以！
我现在就把整份 “言语理解·最核心 10 页笔记” 改写成 超级好读、超级上头、越读越想读的版本。

你会感觉自己像在刷一部节奏飞快、剧情紧凑的爽文。
背题不累，反而会上瘾。

🌈 《言语理解爽文速记版 · 从入门到成仙只需10页》

（看完升天、考试乱杀那种）

🟦 第 1 页：言语理解江湖总纲 · 不学你会哭

欢迎入坑“言语理解”的江湖。

这里有四大流派：
中心派、逻辑派、搭配派、排序派。
但无论你最终想练哪一门绝世神功……

👉 有一个至高口诀必须先背：
转折大于一切，结论永远至尊。

比如作者说：

“虽然我很胖，但是我喜欢我自己。”

你千万别把“我很胖”当重点。
重点是：
👉 我：就：是：爱：自：己。

说穿了这是“言语理解世界”的武林盟主法则：
转折之后皆真理，举例背景皆跑龙套。

🟦 第 2 页：中心理解 · 你要找的，其实是作者的“嘤嘤嘤”

你要找中心句？
很简单：

🎯 作者 95% 的心声只放在三处：

① 开头一句话
② 最后一刀（尾句）
③ 有“因此/所以/可见”的地方

其他什么数据啊、历史啊、专家啊……
全部都是作者为了显得自己厉害一点儿编的铺垫。

真正想说的只有一句：
“你看，我是对的。”

🟦 第 3 页：标题选择 · 你是作者老婆吗？不是就请你概括一点

标题怎么选？
你要做到：
像作者的“贴心小保镖”一样，
用一句话帮 TA 概括全文。

❌ 不要：

说得巨具体：
“1998年猫猫在四川吃了个罐头因此……”

✔ 要：

“猫猫的饮食规律与健康”

标题一定要大气：
大到能盖住全文，
大到你看了觉得：

“哇靠，这标题比文章还像文章。”

🟦 第 4 页：语句填空 · 词语界的爱情匹配

语句填空，其实是：
“给词语找对象”。

主语、语气、情绪必须匹配。

举个例子：

男主：沉稳内敛
女主：炸裂癫狂

这两个放一起就“不搭”。
选项也是一样：

  • 积极+积极 → OK
  • 消极+消极 → OK
  • 严肃+严肃 → OK

但“欣喜 + 崩溃”就离婚预定。

🟦 第 5 页：接语选择 · 承上启下的沟通高手

你要当一个情绪稳定的“话题接龙高手”。

如果上文说：
“小明走进教室。”

你不能突然说：
“而企鹅在南极繁殖。”

这就像
别人跟你聊恋爱，你突然说你妈让我回家吃饭。

要点只有三个字：
👉 不 跳 话。

🟦 第 6 页：词句理解 · 代词就是作者偷懒的地方

“这、其、他们” 这些词，
都是作者偷懒不想重复写同一个名词。

比如：

“小明爱学习。
这让老师很欣慰。”

“这” = 小明爱学习
不是小明本人
不是老师
更不是桌子（？？）

考试就是要你戳穿作者的小心思。

🟦 第 7 页：语句排序 · 把故事讲顺就赢了

排序题是什么？

——就是一段话被人拽下来摔碎了，
你现在要把它拼回去。

你需要注意：

✔ 第一句不能出现：

“因此/但是/这/其/他…”
因为它们都属于“我接着刚刚讲”。
但第一句没有“刚刚”。

✔ 最后一句通常是：

“因此/所以/总之/由此可见”
——作者收尾的“逼王时刻”。

你只需要把故事讲顺
考试会直接给你高分。

🟦 第 8 页：细节判断 · 命题人最喜欢骗你的地方

细节判断题的选项：
通常由两种动物构成：

① 变异怪（偷偷换词，比如“部分 → 全部”）
② 幻术怪（文中没说的，它说你说了）

你的任务：
像在看渣男一样审题。

渣男说：
“我睡沙发。”
但你要问：
“沙发是在男生宿舍还是我家？”

正确选项都是“同义复述”
绝不多、绝不少。

🟦 第 9 页：逻辑填空 · 词语的江湖恩怨

逻辑填空不是要你背字典。

它是：

词语 + 逻辑的武侠混合战。

比如：

“因此、所以、导致、从而”
都是因果必杀技。

“甚至、尤其、恰恰、反而”
是情绪大反转的暗器。

你只需要：

  • 找逻辑
  • 看语气
  • 搭配要自然

你基本能乱杀八方。

🟦 第 10 页：篇章阅读 · 读文章的“灵魂捕手”

长文章很吓人？
没事，我教你一个邪术。

✨ 邪术：抓“文章共同讨论对象”

比如五段话一直在说：
“人工智能如何在养老方面发挥作用。”

那么文章中心就是：

👉 人工智能 + 养老 + 评价 / 对策

至于里面那些例子、历史、政策……
你只需要看作：
——作者的背景音乐。

🌟 完结撒花！你已经读完《言语理解爽文·从凡人到飞升 10 页》

你现在已经掌握了言语理解江湖的 95% 诀窍。
剩下 5% 是你的心态和发挥。
但别担心，看得出来你这次一定能行。

如果你愿意，我还能为你做：

✨ “10 分钟上考场前背诵版（超短）”

✨ “考试前一天的佛系心态巩固版”

📘《数量关系》30 分钟极速复习全提炼版

——上考场前最后一遍看它就够

⭐️ 第一部分：数学基础（必会基础概念）

① 数的认识

  • 整数 = 负整数、0、正整数
  • 自然数 = 0 + 正整数
  • 分数：分子÷分母 → 同乘/同除不变
  • 倒数：乘积为 1
  • 小数点移动：右移 ×10/100，左移 ÷10/100

② 整除

  • 2/5：看最后一位
  • 4/25：看末两位
  • 8/125：末三位
  • 3、9：看数字和是否被整除
  • 0 可被任何非 0 整数整除

③ 质数 & 合数

  • 最小质数 = 2（唯一偶质数）
  • 1 既不是质数也不是合数

④ 最大公因数/最小公倍数

  • 使用：分解质因数法 / 短除法
  • 倍数关系：小的是最大公因数，大的是最小公倍数
  • 互质：最大公因数 = 1；最小公倍数 = a×b

⭐️ 第二部分：常考方法（解题核心）

① 代入排除法（考试第一大法）

适用：
✔ 有选项
✔ 多位数/余数/年龄
✔ “可能/不可能”

原则：

  • 问最大 → 从最大代
  • 问最小 → 从最小代
  • 选最容易算的代

② 数字特性法

✔ 奇偶性（必背）

  • 奇±奇=偶；偶±偶=偶；奇±偶=奇
  • 奇×奇=奇；其余×都偶=偶

✔ 整除特性

  • 比例 a:b=m:n（m、n互质）
    → a 是 m 的倍数，b 是 n 的倍数
  • a±b → 是 m±n 的倍数（重点！）

③ 方程法

  • 设未知数 → 建等式 → 求解
  • 优先设所求量
  • 多元 → 优先做“消元”，配合比例、特性

④ 赋值法

适用：

  • 题型满足 A=B×C（有比例但无具体值）
  • 赋 1、10、100 这类好算的值

口诀：
有比例无具体 → 果断赋值来代替

⑤ 十字交叉法（混合平均、溶液必考）

识别：

  • 给整体平均
  • 给两个部分平均
  • 要你求人数/浓度/销量比例

形式：
(A - 总)(总 - B) 对应 A↔B 的比例

⭐️ 第三部分：比例专题

工程问题（必考）

核心：
工作量 = 效率 × 时间

高频技巧：

  • 两人合作效率 = 1/A + 1/B
  • 工程量不变 → 效率、时间成反比
  • 效率比例已知 → 赋值最快

行程问题（必考中必考）

核心公式：

  • s = vt
  • 当路程相等 → 速度比 = 时间反比
  • 平均速度（等距离）：
    2v1v2 / (v1+v2)

特殊：

  • 追及：s = (v快 - v慢)t

  • 相遇：s = (v1 + v2)t

  • 火车过桥：
    总程 = 车长 + 桥长

  • 流水行船：
    顺 = 船 + 水
    逆 = 船 - 水

溶液问题

核心：
溶质 = 浓度 × 溶液量
常考：

  • 蒸发：溶质不变
  • 稀释：溶质不变
  • 混合：溶质量相加
  • 十字交叉：平均浓度题型

经济利润问题

  • 利润 = 售价 - 成本
  • 定价 × 折扣 = 售价
  • 利润率 = （利润 ÷ 成本）
  • 分段计费 → 注意分段临界点

⭐️ 第四部分：计数问题

容斥

两集合：
A+B-交=总
三集合：
A+B+C -（两两交） +（三交）

必须注意：

  • 题目问“仅满足”与“至少满足”不同
  • 不参加=总 - 至少一个参加

排列组合

口诀：

  • 排有序，组无序
  • 多步法 → ×
  • 多情况 → +

技巧：

  • 捆绑法（必须在一起）
  • 插空法（不能相邻）
  • 隔板法（分配类）
  • 错位排列（Derangement）
  • 环形排列：固定一个人减少一阶

⭐️ 第五部分：最值问题

“最不利”构造

题目含：
至少……保证……

求法：
最不利 + 1

核心知识：

  • 最不利 = 尽量让情况“分散”
  • 比如：抽球要凑相同颜色

数列构造

当题目问：
“最多多少”“最少多少”“第几大的值是多少”

步骤：

  1. 排序
  2. 设目标
  3. 极限构造（让别人尽量大/尽量小）
  4. 求和求出 x

⭐️ 第六部分：几何

平面几何

  • 三角形内角和 = 180°
  • 外角和 = 360°
  • 平行四边形、长方形、菱形、正方形性质（必背）
  • 梯形面积 = (上+下)/2 × 高

圆

  • 周长 = 2πR
  • 面积 = πR²
  • 扇形面积 = θ/360 × πR²

立体几何

  • 长方体表面积 = 2(ab+bc+ac)
  • 正方体 = 6a²
  • 圆柱体积 = πR²h
  • 球 = 4/3 πR³

常考技巧

  • 最短路：拉直/翻折/构造直线
  • 相似三角形：边比 = 面积比的平方
  • 等高三角形：底比 = 面积比

⭐️ 第七部分：高频小题型（非常容易见！）

年龄问题

  • 年龄差永远不变
  • 未来年龄 = 现在年龄 + 年数
  • 用代入排除法最快

星期问题

  • 平年 +1；闰年 +2
  • 每 7 天循环
  • 当天 + n 天 = 星期几（n+1 天法）

牛吃草问题

模型：
原草 + 生长 = 牛数 × 天数

适用各种“消耗/补充”类模型：
排队、排水、买票、机器运转

5）看几何 + 高频小题型（5 分钟）

你要哪一种？

两人合作效率 = 1/A + 1/B 的含义

假设：

甲单独做完一项工程要 A 天

乙单独做完同一项工程要 B 天

那么：

甲一天能做 1/A（整个工程的多少）

乙一天能做 1/B

所以两人一起干时：

一天能做 = 甲一天做的 + 乙一天做的
= 1/A + 1/B
= 两人合作效率

这就是公式的来源。

好！我现在用 史上最好懂、最好记、最好笑 的方式，把“十字交叉法”讲得让你完全停不下来，笑着懂、秒会用。

🎭 十字交叉法是什么？

一句话版本：

当两个东西混在一起，结果在两者之间时，用十字交叉找人数（或量）比例。

比如：

  • 两杯不同浓度的盐水混一起
  • 两种不同价格的苹果混一起
  • 两种不同分数的班级混合求平均
  • 两种速度的路段混合求整体平均

都能用！

十字交叉就是一种：

把两个东西“比一下离平均差多少”，差得多的人少一点，差得少的人多一点。

这就是它的灵魂。

🍹 来一个有趣的比喻（你永远忘不掉）

把两个浓度不同的盐水想成——

两个性格不同的朋友：苦盐哥（浓度高） 和 清水弟（浓度低）

他们俩一起冲饮料，中和成某个“中间浓度”。

  • 清水弟：淡得离谱
  • 苦盐哥：咸得抓狂
  • 平均浓度：刚刚好

十字交叉法干的事就是：

看谁离“刚刚好”更远，就让他少来一点！

这是不是超级合理？
咸得太狠就少来点，不然咸死。
淡得太过就多来点，不然寡淡。

⭐ 十字交叉的公式（秒懂）

画一个十字：

比例 = 右上 : 左下
即：

两者数量比 = (A - 总) : (总 - B)

🍵 秒懂案例（最经典的盐水）

高浓度：20%
低浓度：10%
混合后：15%

画十字：

比 = 5 : 5 = 1 : 1

是不是很合理？
20% 和 10% 离 15% 一样远 → 用量一样。

😭 再来个容易笑出来的例子（极端版）

你有两位“盐水大哥”：

  • 100% 盐（盐粒本体）
  • 10% 盐水

要调成 11% 盐水。

画十字：

比例 = 89 : 1

哈哈，这非常合理：

100% 的太咸了，离目标太远，所以只能来一点点
10% 的比较接近，多来点

你看，十字交叉其实是完全符合直觉的。

🚀 一句话记忆法（最精髓那一句）

谁离目标越远，谁的量就越少；离得近的量越大。

就是这么简单。

🎯 十字交叉适合的题型

只要出现：

✔ 两个已知“平均”（浓度、价格、分数、效率……）
✔ 一个混合平均
✔ 要求比例/数量

就能用。

典型例子：

  • 溶液浓度混合
  • 价格混合（苹果 3 元/5 元混卖平均 4 元）
  • 平均分数混合（两班平均推总体平均）
  • 效率混合（两种速度的平均速度）※特定情况可用

🌟 3 个十字交叉的高频技巧

① “平均值一定在两者之间”

否则不能用十字交叉。

出现下面情况就不能用：

  • 平均 > 最大
  • 平均 < 最小
  • 平均 = 其中之一（直接用比例法）

② 比例求数量：把比例乘到同一个总量上

比如比例 3:2，总量 50 →
3+2 = 5 份 → 每份 10 → 3 份=30，2 份=20

③ 平均题是十字交叉的绝对主场

遇到这些词就条件反射：

  • “混合一起”
  • “平均价格”
  • “平均分”
  • “平均工资”
  • “混合溶液浓度”
  • “总体平均多少”

十字交叉稳稳拿下。

🔥 用一句神总结结束：

十字交叉 = 看两边离目标差多少 → 差得多用得少，差得少用得多。
比例 = 两条斜线上的差值。

这句话你如果记住了，十字交叉这辈子彻底通了。
