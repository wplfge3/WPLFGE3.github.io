技法28 归纳推理”主题一致”速选法
一、主题一致
主题是指一段论述或者一段表述的中心意思，一般通过题干的主题词来定位，分析题干的主题是解题的核心，在解题过程中，要辅助分析题干的逻辑层次和并列的意思表达。

主题一致原则，是要求依据题干可以推出的结论，要和题干表达的中心意思保持一致，据此来判定选项是否为正确答案。

二、违反”主题一致”的表现
违反主题一致原则，选项会出现的典型错误主要有：主体客体偷换，逻辑关系偷换。

技法29 归纳推理”从弱原则”竞选法

从弱原则，也叫做可能性优先原则，是指当多个选项与题干主题一致时，断定程度较弱的选项，一般是优先选择的的正确答案。

不选绝对的，符合从弱原则的选项，一般会出现的关键词是有时、有些、可能、未必、相似等。

技法30 原因解释之”违背常理”

一、原因解释
原因解释属于可能性推理。这种题型，要求考生在题干中寻找到不合常理的现象，然后通过其潜在的原因，使不合理变合理

二、原因解释之”违背常理”

所谓的”违背常理”，指题干先给出一些事实，根据这些事实，很容易根据”常理”得到某些结论，但是题干给出的”违背常理”的结构，故需要解释

常见的解释方式为给出新的理由来解释题干现象

技法31 原因解释之”违背预期”

一、原因解释
原因解释属于可能性推理。这种题型。要求考生在题干中寻找到不合常理的现象，然后通过分析其潜在的原因，使不合理变得合理

二、原因解释之”违背预期”

所谓”违背预期”，指题干给出一项政策，制定该政策的”预期”是为了解决某问题，但是结果并未解决问题，甚至让问题变得更严重

常见的解释方式为指出该政策会导致反效果

技法32 原因解释之”考虑不周”

一、原因解释属于可能性推理，在题干中寻找到不和常理的现象，然后分析其潜在的原因，使不合理变得合理。

二、原因解释之”考虑不周”

所谓”考虑不周”，指题干给出的一些数据，但是由于没有考虑该数据的基数范围，直接得出一些结论

常见的解释方式指出原始数据的基数与结论的差异

技法34 增加论证之举例子

众多加强方式中，举例子加强出现的情况居多，需要了解到举例子是否有陷阱，以及其加强力度如何。

1，举例子加强的陷阱

选项中出现举例子的选项，需要判定例子中的对象和论点是否具有相关性。 否则大部分情况应为”无关选项”

例如：论点：梨中有水分

错误举例子：蔬菜中的白菜具有水分
解析：梨和白菜不具有相关性，无法通过举例进行加强。

2，加强力度如何？

举例子其实是从生活中相似的事物有类似或者相同的现象来证明原论点中主体也具有相同的现象

一、等差数列（Arithmetic Sequence）
定义：数列中从第二项起，每一项与前一项的差是常数，这个常数叫公差 d。
例如：2, 5, 8, 11, 14, …（公差 d = 3）
通项公式：aₙ = a₁ + (n − 1)d
前 n 项和公式：Sₙ = n/2 (a₁ + aₙ) 或 Sₙ = n/2 [2a₁ + (n − 1)d]
常见考点：
① 已知三项求公差：d = (a₃ − a₁)/2
② 等差中项：若 a, b, c 成等差，则 b = (a + c)/2
③ 插项问题：在两数之间插入若干等差数，使其成为等差数列。
④ 求和技巧：可用配对法或首末项相加法。
二、等比数列（Geometric Sequence）
定义：从第二项起，每一项与前一项的比是常数，这个常数叫公比 q。
例如：2, 6, 18, 54, …（公比 q = 3）
通项公式：aₙ = a₁ q^(n − 1)
前 n 项和公式：当 q ≠ 1 时，Sₙ = a₁ (1 − qⁿ)/(1 − q)；当 q = 1 时，Sₙ = n a₁
常见考点：
① 已知三项求公比：q = √(a₃ / a₁)
② 等比中项：若 a, b, c 成等比，则 b² = ac
③ 无穷等比数列：若 |q| < 1，则 S∞ = a₁ / (1 − q)
④ 错位相减法：求复杂等比数列和时常用 qSₙ − Sₙ。
三、综合与易错点
类型 关键点 常用公式 / 方法
等差数列求项 公差固定 aₙ = a₁ + (n − 1)d
等差数列求和 对称求和 Sₙ = n/2 (a₁ + aₙ)
等比数列求项 公比固定 aₙ = a₁ q^(n − 1)
等比数列求和 错位相减法 Sₙ = a₁ (1 − qⁿ)/(1 − q)
无穷等比数列 收敛条件 |q| < 1 S∞ = a₁ / (1 − q)
四、总结记忆法
等差：加法规律 → 用减法求 d，用加法求 Sₙ。
等比：乘法规律 → 用除法求 q，用乘法或相减法求 Sₙ。

三、整理要点
(一)整理要点方法
1.前化表达;精简表达、副除修饰信息、例子信息，重复信息、播述性词语2.裁范表达:对口摄化表述进行改写，故成政府规范用语
3.逻辑加工:对相同属性的信息进行整合、整理成以动宾结构为主的中心句。
(二)归类整理方法
1.主体分类法
根据不同的主体，可以按个人、企业、政府、社会、国家等进行归类。一般适用于材料有三个及以上不同主体的题目。
(1)个人:道德、利益、素质、观念。
(2)企业:工业、设备、投人、创新、品牌。
(3)政府:认识、法规、制度、投入(资金、人员)、监管、执法、协调、领导、重视、机制、体制、规划。
2.措施分类法
按胲讫Ë不慧飒鉺滑同的做法进行归类，例如宣传、制度、教育、监管、投入、创新等，一般用于提炼经验、做法等题目。
3.维度分类法
按政治、经济、生态、文化、社会等分类，一般用于提炼优势、作用等题目。
4.关键词分类法
按题于关键词进行分类。一般用于题目题干较长、信息较多，有明确提示分类依据的题目。
5.文段分类法
按不同材料或者段落进行归类整合;一般用于材料内容较少、材料落内部逻辑分类非常清晰的题目。
按定义、表态、表现、原因、意义、问题、对策等不同材料要素进行归类，一般用于题目要求答题元素较多的复合型题目中，比如复合型解决问题题、复合型综合分析题，或者应用文写作题
6.要素分类法

四、书写答案
(一)标记序号
为了让表述更有条理、更清晰，可以使用序号分条列出，但要注意序号之间的层级关系。
序号层级由大到小:一>(一)>1.>(1)>①。
建议书写序号时尽量使用“(1)”“①”来代替“1.”,带有括号、圆圈的这类序号在答题纸上会更明显，缩短了阅卷人寻找答案要点的时间。
注意:不是所有题型都需要标记序号，比如文章写作题、部分应用文写作题可以不标记序号，甚至有些应用文写作题也可以使用“首先”“其次”等表示逻辑的词。
(二)卷面整洁
申论卷面力求字迹清晰、工整，平时做题练习时要避免写连笔。如果在书写答案时出现错别字，划去错别字，在旁边方格中继续书写对的内容，尽量一笔勾画，线、斜线均可，不可以用画圈或涂黑的方式处理。
